Scrapy命令行:
 - scrapy -h

startproject #创建一个新工程
 - 格式:scrapy startproject <name> [dir]

genspider #创建一个爬虫
 - 格式:scrapy genspider [options]<name><domain>

settings #获得爬虫配置信息
 - 格式:scrapy settings [options]

crawl #运行一个爬虫
 - 格式:scrapy crawl<spider>

list #列出工程中所有爬虫
 - 格式:scrapy list

shell #启动URL调试命令行
 - 格式:scrapy shell [url]
